Quiero cambiar la lógica de generación para que no dependa de temas específicos, sino del tipo de contenido educativo.

Problema actual:
El sistema trata el tema como una palabra a repetir (“ecosistema”, “sustantivos”, etc.) y genera contenido genérico o incorrecto.

Objetivo:
Que el sistema interprete el tema como un CONCEPTO y lo desarrolle correctamente según su tipo, sin necesidad de cargar temas manualmente.

Reglas obligatorias:

1. NO TRATAR EL TEMA COMO TEXTO

- No repetir el nombre del tema como frase
- No usar estructuras como:
  “trabajar el tema de…”
  “aplicar correctamente…”

El tema debe interpretarse como concepto, no como palabra.

2. CLASIFICAR AUTOMÁTICAMENTE EL TEMA

El sistema debe identificar a qué tipo pertenece:

A) Lenguaje básico:
(letras, sílabas, fonemas)
→ actividades: reconocer, completar, escribir

B) Gramática:
(sustantivos, verbos, adjetivos)
→ actividades: identificar, clasificar, producir

C) Comprensión / texto:
(texto narrativo, cuento, noticia)
→ actividades: analizar, interpretar, escribir

D) Matemática:
(fracciones, suma, multiplicación)
→ actividades: resolver, representar, aplicar

E) Ciencias naturales:
(ecosistema, animales, cuerpo humano)
→ actividades: clasificar, explicar, relacionar, dar ejemplos reales

F) Ciencias sociales / historia:
(independencia, revolución, sociedad)
→ actividades: comprender, causa-consecuencia, línea de tiempo

3. DESARROLLAR EL CONCEPTO

Una vez identificado el tipo:

- expandir el tema en ideas enseñables
- generar contenido real
- usar ejemplos concretos

Ejemplo:
Tema: “ecosistema”
→ no repetir “ecosistema”
→ generar:
- seres vivos y no vivos
- relaciones entre organismos
- ejemplos del entorno
- actividades de clasificación

4. COHERENCIA TOTAL

Todo debe coincidir:
- tipo de contenido
- materia
- nivel educativo
- actividades

5. EVITAR GENERALIZACIÓN

No usar estructuras que funcionen para cualquier tema.
Cada contenido debe sentirse específico.

6. RESULTADO FINAL

El contenido debe parecer:
- pensado por un docente
- con sentido pedagógico real
- aplicable en el aula
- basado en el concepto, no en la palabra
